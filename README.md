# Anomaly-Based-Intrusion-Detection-Using-CICIoT2023
Anomaly-Based Intrusion Detection in IoT Networks: A Comparative Study of Traditional Machine-Learning and Deep-Learning Techniques Using CICIoT2023

This repository contains the implementation for a comparative study of
traditional machine-learning and deep-learning techniques for anomaly-based
intrusion detection using the CICIoT2023 dataset.

## Research Aim

The project evaluates whether traditional machine-learning and deep-learning
models differ significantly in their ability to identify benign and malicious
IoT network traffic.

## Dataset

The project uses the CICIoT2023 dataset produced by the Canadian Institute for
Cybersecurity.
